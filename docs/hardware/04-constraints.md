## PCB/DFM/DFT

- 4 层 1.6mm FR-4、标称 1oz、沉金；L2 保持连续 GND。
- 最终主板为 `112×68mm`、4 层、1.6mm 长方形单板；以左上角为 `(0,0)`，屏幕投影为 `x=8.715～103.285mm、y=3.56～64.44mm`，即 `94.57×60.88mm` 居中安装在 PCB 背面。40001 手册中的 `108.1×61.74mm` 与孔位属于 34Pin 转接模块，不得作为本板数据来源。
- J1=`KH-TYPE-C-16P` 固定在左侧边几何中点并朝外，EasyEDA 中心为 `(4.318,34.0)mm、rotation=90°`；U5 位于右侧，天线端完整伸出屏幕金属投影。U1、R1/R2、C19 保持靠近 J1，后续布线不得为了躲避飞线破坏 USB ESD 先泄放、CC 下拉或本地去耦的相对关系。
- 4 个 M3 安装孔均为非金属化 `Ø3.2mm`，中心 `(4,4)`、`(108,4)`、`(4,64)`、`(108,64)mm`；每孔周围 `R4mm` 同时禁止器件、走线、填充、覆铜和内电层。
- WROOM-1 天线端伸出 LCD 金属背板投影，所有层净空，远离喇叭磁体、U.FL/SMA 和大电流环路。
- USB D+/D- 按 90Ω 差分、短、等长、少过孔；ESD 在连接器侧，串阻在 ESP32 侧。
- SY8113I 的 CIN、CBS、功率环路、LX 与电感必须极短；两路 Buck 和 Class-D 与 GNSS RF 分区。
- RF_IN 到 U.FL 为 50Ω，VCC_RF 通过 47nH 馈电；R34=0Ω 默认装配，C32/C33=1pF C0G 强制 DNP。D4=`PESD5V0R1BSFYL` 强制 DNP，后续替代料必须明确支持带 3.3V 直流偏置的 RF 馈线、结电容不高于 0.2pF，并完成插损、定位和天线 OPEN/OK/SHORT 验证。
- U6=`ATGM336H-F8N-76` 的 XY 按原厂/LCSC/焊盘图共同支持的 `10.1×9.7mm` 封装；资料对高度存在 `2.1/2.4mm` 冲突，布局与结构先按 `2.4mm` 最坏值预留，首批实物测高后再冻结 3D 间隙。
- HT517 本地默认只装 C34=100nF、C35=10µF；C36=470µF/16V 强制 DNP。FB1/FB2 为 0805 0Ω 默认装配，磁珠替代需额定不低于 2A、低 DCR 且成对验证；C37/C38=1nF C0G 强制 DNP。
- J2 固定为 `FPC-0.5HF-40PWBH20 / C3446044`，中心 `(37.971419,33.01)mm、-90°`；J3 固定为 `FH12-6S-0.5SH(55) / C202118`，中心 `(15.214127,53.56)mm、+90°`。两者均竖放、入口朝左；J2 入口距槽右边约 `1.0mm`，J3 入口距槽右边约 `0.3mm`。
- J6 固定为 `2.0-5P WZ-D / C41361332`，PH2.0 卧式弯插通孔座，中心 `(16.51,58.42)mm、0°`，位于左下板边内侧且开口朝下/板外；Pad1～Pad5 从左到右依次为 `GND、3V3_EXP、EXP_GPIO1、EXP_UART_TX_GPIO43、EXP_UART_RX_GPIO44`。J6 本体、焊盘和 R38～R41 均在板框内，严格 DRC 无 Clearance/Keepout 错误；左下 M3 的 `R4mm` 全层禁布仍保持。该连接器库料标注“不耐温”，只允许 SMT 回流后手工或选择焊装配。
- R38=`0Ω/0805` 位于 `(10.922,47.752)mm`，把 `3V3` 隔离为 `3V3_EXP`；R39～R41=`33Ω/0603`，中心分别为 `(14.732,47.752)`、`(18.542,47.752)`、`(22.352,47.752)mm`。J6 按壳内非热插拔扩展口使用，3.3V 只允许主板输出，建议持续负载不超过 300mA，禁止外部反向供电；改为外露接口前必须补做 ESD、浪涌、短路、线缆 EMI 和防水评审。
- 2026-07-20 为降低穿线勉强和槽长公差风险，40Pin 穿线槽加长为竖向 `3×23mm、R1.5`，中心 `(32,33.01)mm`；6Pin 穿线槽加长为竖向 `3×6mm、R1.5`，中心 `(8.715,53.56)mm`。每槽外至少 `0.5mm` 区域禁止走线、填充、覆铜和内电层，对应包络为 `4×24mm、R2` 与 `4×7mm、R2`；左、下侧和 6Pin 上侧设 `1mm` 器件禁布带，40Pin 上侧因 C16 外框设 `0.9mm`，右侧只为对应连接器入口保留明确例外。
- 40Pin FPC 总长 `42.1mm`。根部到槽左边 `21.785mm`，穿槽/翻面保守段 `4.6mm`，槽到 J2 入口 `1.0mm`，插入段 `4.5mm`；加入 `R1.5mm` 回折和两侧各 `1mm` 释放段后，设计路径 `38.597mm`、余量 `3.503mm`。6Pin FPC 总长 `11.05mm`，名义路径 `7.9mm`、保守路径 `9.4mm`，最坏余量仅 `1.65mm`。`R1.5mm` 回折是工程假设，必须用屏幕实物或供应商结构图确认。
- C16 使用 `CAP-TH_L16.0-W8.0-P11.5-D0.6` 横卧封装，本体约 16×8mm、脚距 11.5mm；避开射频、开关节点和安装孔，并用中性无腐蚀胶抗振固定。
- TP1～TP22 使用 `TESTPOINT-SMD_BD1.5_P70-5000045R` 的单个无孔圆形 SMD 焊盘，实际铜盘约 2.00mm；全部放到底层、保持 2.54mm 针床节距、整板沉金、不装器件并从钢网开口中排除。
- Pogo 点覆盖 USB_VBUS、VIN_PROT、3V3、5V_AUDIO、GND、EN、BOOT、USB D+/D-、UART0、GNSS UART/1PPS、eFuse `SPGD/nFAULT`、GNSS ON/OFF/nRESET 与 VBAT 充电/保持网络；USB D+/D- 支路必须短且对称，PCB 阶段复核差分间距和支路 stub。
- 原理图→PCB 交互导入及 J6 扩展更新均已保存、关闭、重开。当前回读为 132 个器件、411 个焊盘、89 个网络、2 个机械槽、12 个机械禁布区域和 4 个 M3 NPTH；22 个 Pogo 全部在底层。电气走线、过孔和覆铜仍为 0，严格 DRC 只有 `Connection Error (374)`；Clearance 与机械 Keepout 错误均为 0，但这不代表 PCB DRC 通过。
- 当前功能域位置为：GNSS/U.FL 左上、J1/USB ESD/CC 下拉位于左侧边中点、输入/eFuse 左下、J6 扩展口位于左下板边、双 Buck 中部、音频下中部、ESP32 右侧；J2/J3 与两个槽位于屏幕 FPC 根部对应高度。投板前必须用 `preview/pcb-final-112x68-fpc-mechanical-1to1.pdf` 以 100% 实际尺寸打印，先校准 50mm 标尺，再核对 Type-C 插拔、M3 螺钉、J2/J3/J6 线束、FPC 金属锐边和 ESP32 天线遮挡。
- 嘉立创EDA原生 `pcb_Document.autoRouting()` 已在 J6 更新后再次诊断：即使请求只包含 7 条低速扩展网络并限制 Top/Bottom，后端仍返回 `success=false、totalNetsCount=89、successNetsCount=0、duration=0`，并把全部 89 个网络列为失败；调用前后铜线/过孔均为 `0/0`。因此原生自动布线后端当前不可用，结果不得作为 PCB 输入。
- 本 GPIO 表属于新板接口合同；当前旧 ESP32 固件 GPIO 仍以 `.trellis/spec/backend/hardware-build-flash.md` 为准，后续迁移任务必须同步修改代码权威。

# esp32BmsGps 车机硬件方案

- 日期：2026-07-19
- 阶段：7 - 决策门

## 已确定

- ESP32-S3-WROOM-1-N16R8、4.0 寸 ILI9488/FT6336U、8 位 I80。
- ATGM336H-F8N-76 + 外置双频有源天线 + U.FL 到防水 SMA。
- HT517SPER + 4Ω/5W 喇叭。
- USB 5V/专用12V、SGM2536FR、双 SY8113I、3A Fuse、SMBJ17A。
- J2=`FPC-0.5HF-40PWBH20 / C3446044`，J3=`FH12-6S-0.5SH(55) / C202118`，均已绑定精确 EasyEDA 封装。
- C16=`CHP5R5L105R-TW / C2925920`，5.5V/1F 横卧 H 型，已绑定 16×8mm、11.5mm 脚距封装。
- TP1～TP22 已替换为可转 PCB 的单引脚约 2.00mm 无孔 SMD 裸盘，BOM 排除。
- D4=`PESD5V0R1BSFYL` 保留封装但强制 DNP；该料不作为带 3.3V 偏置 GNSS RF 馈线的已验证保护器件。
- C36=`470µF/16V` 保留封装但强制 DNP；5V_AUDIO 默认只装 100nF+10µF 本地去耦。
- 4 层沉金单主板，不加 TF、CAN、RS485、环境光传感器和外部用户按键。
- 最终 PCB 为 `112×68mm` 长方形并位于 4.0 英寸屏幕后方；屏幕投影 `94.57×60.88mm` 居中，4×M3 NPTH、两个竖向穿线槽和槽外机械禁布区均已进入 EasyEDA。
- USB-C 与 ESP32 分居主板相对两侧：U5 在右侧且天线端伸出屏幕投影，J1 固定在左侧边中点 `(4.318,34.0)mm、90°` 并朝外。
- 2026-07-20 用户明确增加穿线余量：40Pin 槽中心 `(32,33.01)mm`、尺寸 `3×23mm`；6Pin 槽中心 `(8.715,53.56)mm`、尺寸 `3×6mm`。两槽长边均与 FPC 宽度方向一致，圆端 `R1.5`；中心、连接器位置和槽宽不变。
- J2/J3 均竖放且入口朝左；40Pin 按 `R1.5mm` 回折假设后的设计路径为 `38.597mm/42.1mm`，余量 `3.503mm`；6Pin 保守路径为 `9.4mm/11.05mm`，余量 `1.65mm`。
- 新增 J6 壳内扩展口：`2.0-5P WZ-D / C41361332`，针序固定为 `GND/3V3_EXP/GPIO1/GPIO43_TX/GPIO44_RX`；R38=0Ω/0805、R39～R41=33Ω/0603。J6 中心 `(16.51,58.42)mm`，本体与电阻均在板内且严格 DRC 无 Clearance/Keepout 错误。

## 待后续阶段确认

- ATGM336H-F8N-76 首批实物高度；XY 已按 `10.1×9.7mm` 收敛，Z 向在 `2.1/2.4mm` 资料冲突关闭前按 2.4mm 预留。
- U.FL、RF ESD、磁珠和电感的最终采购料号。
- 1:1 图与屏幕/外壳实物的最终叠加结果；若 Type-C、M3、J2/J3 锁扣、FPC 回折或天线净空出现干涉，必须回到 PCB 机械布局，不得仅修改外壳强行装配。
- J6 插头/线束与外壳底边、左下 M3、屏幕后盖的实际装配余量；当前只完成 PCB 内部摆放与 DRC，仍须实物插拔确认。

## 高风险

| 风险 | 等级 | 影响 | 验证/应对 |
|---|---|---|---|
| 屏幕随机替换控制器/触摸/接口 | 高 | 首板无法点亮或触摸失效 | 采购书面锁定并核对来料 |
| 背光额定参数缺失 | 高 | 过流、亮度不足或寿命降低 | 可调阻值+样品温升/电流测试 |
| 12V USB-C 非标准 | 高 | 误插外部设备 | 专用线束、醒目标识、禁止直连手机 |
| eFuse/TVS 28V边界 | 高 | 18V误接或脉冲损坏 | SGM2536完整手册+受控脉冲验证 |
| 低价模组/散货 | 中 | 容量、批次、寿命不一致 | 每批抽检 Flash/PSRAM/无线/温升 |
| RF/喇叭/开关电源互扰 | 中 | GNSS灵敏度、Wi-Fi、音频噪声下降 | 分区、回流控制、近场与实机测试 |
| D4 的 DC-biased RF 适用边界未关闭 | 高 | 有源天线偏置与 v2/v3 数据手册使用边界存在风险 | D4 强制 DNP；另选明确支持直流偏置且 `Cj≤0.2pF` 的器件 |
| C36 启动充电浪涌 | 高 | 可能触发 2.38A eFuse 限流、掉压或重启 | C36 强制 DNP；试装前完成 5V/12V 启动与反复通断验证 |
| 40Pin 根部回折半径假设不成立 | 高 | 42.1mm 排线余量不足、疲劳开裂或压到金属锐边 | 以 R1.5+两侧各1mm计算仅余3.503mm；实物确认供应商最小半径并加绝缘膜 |
| 6Pin 最坏余量仅1.65mm | 高 | J3 无法完全插入、锁扣不闭合或长期受拉 | 1:1+实物验证；锁扣闭合后不得有持续拉力，否则移动 J3/槽位 |
| J6 反灌/启动串口争用 | 中 | 扩展板导致主板反向供电、启动日志误动作或 GPIO 强驱冲突 | 3V3 仅输出且≤300mA；GPIO43 启动日志实测；外设不得在启动时强驱；外露使用前重做保护评审 |

## Gate 结论

R1 原理图电气门已通过：P1～P6 真实器件数分别为 `21/21/12/22/16/35`，合计 127；P5/P6 的 3+5 个空名端口已重建并经保存重开确认。严格 ERC 为 `fatal=0、error=0、warn=6、info=151`。6 条 warning 中，3 条是相同网络名的重复标记，3 条是器件属性与供应商编号标准化建议；均已逐条导出，不代表电气错误，也不得宣称为“无警告”。

PCB 同步门也已通过：交互导入后保存、关闭、重开，回读为 127 个器件、394 个焊盘、84 个网络；原理图与 PCB 均为 127 个器件，缺失/多余器件、重复位号/Unique ID、位号错配和封装错配均为 0。同步后快照 `post-pcb-import-20260718-235800.eprj2` 的 `PRAGMA integrity_check=ok`，SHA-256 `15a359c28dec21436136e61cefd65a0048c28e81e3d6621a9785bb93e30ab012`。

PCB 机械布局门已部分推进：EasyEDA 已建立 `112×68mm` 板框、`94.57×60.88mm` 屏幕投影、4×M3 NPTH、两个竖向槽和 12 个机械禁布区域；127 个器件全部位于板内，TP1～TP22 全部在底层 2.54mm 阵列。保存、关闭、重开后回读为 127 个器件、398 个焊盘；快照 `post-fpc-orientation-path-verified-20260719-133816.eprj2` 的 `PRAGMA integrity_check=ok`，SHA-256 为 `5622b9b3b08d7729af41429aee93f5165884402ae9ab64f821112110aafc7547`。1:1 合同图已输出为 `preview/pcb-final-112x68-fpc-mechanical-1to1.{svg,png,pdf}`。

J6 增量已保存、关闭、重开并回读：P3 由 12 个器件/57 条导线增至 17 个器件/64 条导线，整板原理图与 PCB 均为 132 个器件；PCB 为 411 个焊盘、89 个网络。J6、R38～R41 均在板内，严格 PCB DRC 无 Clearance/Keepout 错误，只剩未布线的 `Connection Error (374)`。严格 ERC 为 `fatal=0、error=0、warn=9、info=153`：5 条 warning 是完全相同网络名的重复标记（`USB_VBUS、FB_3V3、FB_5V_AUDIO、DBG_UART_RX_GPIO44、DBG_UART_TX_GPIO43`），4 条是器件属性/供应商编号标准化建议；均非跨网短接或悬空错误，但不得宣称为“无警告”。

嘉立创EDA原生自动布线在 J6 更新后再次失败：请求只布 7 条低速扩展网络时，后端仍错误地把全部 89 个网络列入并返回 `success=false、successNetsCount=0、duration=0`；线段/过孔保持 `0/0`。修改前快照 `esp32BmsGps-Hardware-R1.pre-expansion-j6-20260719-155506.eprj2` 的 SHA-256 为 `5622b9b3b08d7729af41429aee93f5165884402ae9ab64f821112110aafc7547`；最终快照 `esp32BmsGps-Hardware-R1.post-expansion-j6-20260719-162657.eprj2` 的 SHA-256 为 `009d746ed937bc157982c8207f342ccad0836a459489d9dd5b46465bfe6b4a4b`，两者 SQLite `integrity_check=ok`。

该状态仍不是可投板布局：电气走线、过孔和覆铜均为 0；机械方向/坐标已进入工程，但屏幕实物的折弯、锁扣、触点面、绝缘、J6 线束和天线遮挡尚未关闭。正式投板还需完成 4 层/网络规则、关键回路收敛、布线覆铜、Pogo 钢网排除、飞线清零、严格 DRC、3D/DFM 和 EVT 风险闭环；在此之前统一标记 `PROVISIONAL / DO NOT FAB`。

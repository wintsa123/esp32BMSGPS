## 高风险闭环

背光、FPC 方向/折弯、低价 ESP32 模组批次、GNSS 天线频段、SGM2536/SY8113 热表现均不得仅凭原理图或 1:1 图宣布通过，必须保留实测记录。40Pin 的 `R1.5mm` 回折半径是工程假设，设计余量仅 `3.503mm`；6Pin 保守余量仅 `1.65mm`，二者都属于投板前实物门。J6 只按壳内非热插拔口放行，300mA 是验证上限而不是主板对任意扩展板的供电承诺；反灌、短路、GPIO43 启动日志和后焊抗振均需实测。D4 与 C36 当前均为强制 DNP；任何试装或替代都会使相应 RF/启动验证重新打开。PCB 已有 `112×68mm` 板框、最终槽/孔/禁布区和功能域布局，但电气走线、过孔和覆铜仍为 0；严格 DRC 的 `Connection Error (374)` 只证明未布线，不能计入投板通过证据。嘉立创EDA原生自动布线在 89 个网络上仍以 `duration=0` 失败，未生成任何线段或过孔。

# 显示与输入驱动矩阵

## Goal

建立能力驱动的显示/输入后端，使同一核心可运行完整彩屏仪表或 128x64 OLED 精简界面。

## Requirements

- 支持 ILI9341/ILI9488/ST7789/ST7796/GC9A01/NT35510、RGB+ST7701、SSD1306/SH1106、RM67162 和 P4 ILI9881C/EK79007 MIPI-DSI。
- 输入后端支持触摸、按键和旋转编码器，并从 board/profile 获取 GPIO 和电气能力。
- 彩屏保留当前完整仪表、设置、动画和适用投屏入口。
- OLED 使用独立精简状态页与分页设置，不链接彩屏动画和投屏页面。
- 设置页依据编译期能力生成；语言入口只在设置，中文默认，TFT 状态用 ASCII `ZH`/`EN`。

## Acceptance Criteria

- [ ] 每种显示描述都能通过能力/总线/GPIO 配置校验并进入对应基准构建。
- [ ] S3 I80 ILI9488 + FT6336U 初始化、旋转和输入坐标路径有构建/实机证据。
- [ ] 触摸、按键、编码器均映射到统一 LVGL input contract。
- [ ] OLED 镜像不含彩屏动画/投屏页面符号和资源，128x64 布局可读且可分页设置。
- [ ] 所有新增 preview 只写根 `preview/`。

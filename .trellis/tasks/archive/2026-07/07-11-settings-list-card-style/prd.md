# 设置页列表卡片样式

## Goal

调整 TFT 设置根页和二级设置页的列表视觉层级，使列表集中在卡片内，并保持各行内容对齐稳定。

## Requirements

- 设置页列表外的页面背景保持黑色。
- 五个设置选项放在一个有页面边距的统一卡片内。
- 热点、蓝牙、保护板、系统和关于本机的二级列表使用同样的统一卡片样式。
- 卡片内列表行使用灰色背景，行之间用现有设置边框色分隔。
- 每行的标题/副标题文字块和右箭头分别垂直居中。
- 保持现有点击、滑动、进入详情页和横竖屏行高逻辑不变。

## Acceptance Criteria

- [x] 设置根页和二级页卡片外区域显示为黑色。
- [x] 各页设置选项视觉上属于统一的灰色卡片列表。
- [x] 标题/副标题整体与右侧箭头/开关在每行内垂直居中。
- [x] 横屏和竖屏均无内容溢出、重叠或点击回归。
- [x] ESP-IDF 构建通过，GitNexus 变更检测只报告预期的设置 UI 符号/流程。

## Out Of Scope

- 不修改设置项文案、图标、操作行为或功能型子页（扫描列表、滑块、校准、二维码）布局。
- 不新增布局抽象、主题系统或额外依赖。

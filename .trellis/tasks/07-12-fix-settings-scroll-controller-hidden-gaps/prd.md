# 修复设置滚动与控制器隐藏项留白

## Goal

修复 TFT 设置页顶部标题无法稳定收缩、列表无法滚动到底部，以及控制器页面关闭后卡片保留隐藏行空位的问题；同时将控制器蓝牙选择改为复用保护板扫描子页，并将轮胎与传动比设置改为有范围的滚轮选择。

## Background

- 设置根页和详情页共用 `settings_detail_header`，收缩动画同时修改标题位置、根页/详情页顶部 padding 和左侧返回手势区域。
- 当前动画期间的 padding 变化会触发 LVGL 重新计算滚动位置；由布局变化产生的滚动事件被当成用户向上浏览，导致标题刚收缩又展开，并再次遮住列表底部。
- 控制器设置卡片当前先按全部 6 行创建，再在关闭控制器页面时跳过“控制器连接”和“蓝牙选择”并事后缩短卡片，初始几何会留下隐藏项对应的空白区域。
- 控制器“蓝牙选择”当前在主卡片内启动扫描并追加候选行，与已有的保护板蓝牙选择子页交互不一致。
- 当前配置只保存换算后的轮胎周长（mm）和传动比百分定点数，轮胎周长与传动比使用 `- / +` 步进页；不能表示用户直接选择的三段轮胎规格。
- LVGL roller 在当前有效 `sdkconfig` 中未启用，实现前需同时更新可复现的默认配置和有效配置。
- GitNexus 修改前影响分析将 `settings_navigation_set_hidden` 标记为 CRITICAL（6 个直接调用、19 个上游符号、6 条流程），将 `settings_navigation_apply_offset` 和 `settings_show_controller_detail` 标记为 HIGH；影响集中在设置导航、返回手势和设置快照刷新。

## Requirements

- 设置根页及所有可滚动详情页向下浏览时，顶部标题按现有阈值收缩并保持收缩，不得被动画自身产生的滚动事件立即展开。
- 用户向上浏览或回到列表顶部时，标题仍按现有交互恢复显示。
- 标题收缩后，滚动容器释放出的空间必须可用于内容显示，根页和长详情页最后一行可完整滚动到可见区域。
- 保持现有标题、返回按钮、左边缘返回手势、横竖屏尺寸和页面切换行为不变。
- 控制器页面显示关闭时，只渲染“控制器页面显示”和“控制器类型”两行；同时隐藏“控制器连接”“蓝牙选择”“轮胎周长”和“传动比”，卡片底边紧跟在“控制器类型”下面。
- 控制器页面显示开启时，恢复“控制器连接”和“蓝牙选择”，卡片连续显示六行；扫描候选列表位置继续跟随实际可见行数。
- 点击控制器“蓝牙选择”后进入独立子页，复用保护板蓝牙选择的页面结构、扫描状态、刷新按钮、候选列表、确认弹层、顶部返回和左边缘返回；仅数据源和绑定 action 不同。
- 轮胎规格和传动比设置行默认始终显示；控制器蓝牙未在线时使用 `LV_STATE_DISABLED` 置灰且不可进入编辑页，在线后可编辑。
- 用户不再手工填写轮胎周长，而是通过有界的 LVGL 滚轮选择三段轮胎规格；固件内部换算并继续向车速计算提供周长毫米值。
- 轮胎规格按“轮辋直径（英寸）-扁平比（%）-胎宽（mm）”顺序显示；例如 `12-70-90` 等价于标准轮胎写法 `90/70-12`。
- 轮胎规格滚轮范围为：轮辋 `8–24` 英寸（步进 1）、扁平比 `30–100`% （步进 5）、胎宽 `50–200 mm`（步进 5）。
- 传动比默认为 `1.00`，通过数字滚轮在 `0.50–10.00` 范围内选择，步进为 `0.01`。
- 新配置必须持久化；旧 NVS 中只有周长/传动比时必须有明确兼容或迁移策略，不得导致旧设备启动失败。

## Acceptance Criteria

- [ ] 设置根页连续上拖后标题完整收缩且不会自行弹回，列表最后一项可完整显示。
- [ ] 设置详情页连续上拖后标题完整收缩且不会自行弹回，长列表最后一项可完整显示。
- [ ] 下拖或滚动回顶部后标题恢复；返回按钮和左边缘返回均可正常使用。
- [ ] 控制器页面显示关闭时卡片恰好连续显示两行，底边紧跟“控制器类型”，无隐藏项空位。
- [ ] 控制器页面显示开启时卡片连续显示六行，候选设备列表不重叠且没有额外空白。
- [ ] 点击控制器蓝牙选择后进入与保护板一致的独立扫描页，可刷新、选择、确认绑定，且两种返回方式都可用。
- [ ] 轮胎规格和传动比在控制器页面开启时始终显示；控制器未在线时两行置灰且点击无效，在线后恢复可编辑状态。
- [ ] 轮胎规格编辑页使用三个有界滚轮，显示值、NVS 存储值和换算后的周长一致，重启后保持。
- [ ] 传动比编辑页使用有界数字滚轮，新设备和缺少该 NVS 键的旧设备默认显示 `1.00`，重启后保持。
- [ ] 竖屏与横屏布局均通过检查，现有设置项点击和开关操作无回归。
- [ ] `git diff --check`、ESP-IDF 构建、GitNexus `detect_changes` 和 RFC2217 真机启动验证通过。

## Out Of Scope

- 不改变设置页视觉主题、文案、行高、滚动阈值或动画时长。
- 不重构设置组件体系，不修改控制器遥测协议或首页数据页逻辑。
- 不向 FarDriver 控制器回写轮胎或传动比参数；本功能仅作为本机车速换算配置。

## Open Questions

- “每位数字”是指每个字段一个滚轮，还是每个十进制数位单独一个滚轮，尚未确认。
